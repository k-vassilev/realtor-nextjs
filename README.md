An app for browsing real-estate properties to buy or to rent. The app is fething data from an external API - rapidapi.com. The app is deployed and hosted on vercel.com

## For local install

First, run the dev server:

```
npm run dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the app.

Deployed on vercel and can be seen  here<a href="https://realtor-nextjs.vercel.app/">live</a>
